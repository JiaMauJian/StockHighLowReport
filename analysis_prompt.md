# 台股市場溫度分析 Prompt
> 每次分析時，將此文字連同以下三個檔案一起傳給 Claude：
> 1. `market_data_YYYYMMDD.xlsx`
> 2. `market_analysis_YYYYMMDD.txt`
> 3. `taiwan_market_events.md`

---

## 使用方式

直接複製以下 Prompt，貼到對話框，同時上傳三個檔案。

---

## Prompt

```
我有三份檔案：
1. market_data_YYYYMMDD.xlsx — 台股各指標近 20 年原始資料
2. market_analysis_YYYYMMDD.txt — 當前各指標數值與歷史百分位摘要
3. taiwan_market_events.md — 台股歷史重大事件年表與空頭統計

請根據這三份資料，幫我做一份市場溫度分析，格式如下：

1. 整體判斷
   - 一句話結論（過熱 / 中性 / 過冷）
   - 平均溫度分數與對應區間說明

2. 各指標解讀
   - 60日新低比例
   - 60日新高比例
   - 大盤融資維持率
   - CNN Fear & Greed
   - 上市 / 上櫃 月 MACD OSC
   每項指標說明當前位置相較歷史的意義（2~3 句）

3. 歷史對照
   - 目前最像哪一個歷史時期？
   - 該時期之後市場的走向為何？

4. 核心矛盾與風險提示
   - 指標之間有沒有互相矛盾的訊號？
   - 目前最值得警惕的一個訊號是什麼？

5. 觀察清單
   - 接下來要追蹤哪些指標的變化，才能判斷趨勢是否轉折？
   - 給出具體的門檻數字

注意事項：
- 語氣客觀，不要給明確買賣建議
- 如果指標之間有矛盾，直接說，不要硬湊結論
- 如果我的判斷有偏誤，請直接糾正
```

---

## 補充說明（視情況加在 Prompt 後面）

**如果最近有重大事件：**
```
補充背景：最近發生了 [事件名稱]，請將此納入分析。
```

**如果想做特定比較：**
```
請重點比較目前與 [年份] 的相似程度。
```

**如果想聚焦底部訊號：**
```
請特別分析目前是否有任何恐懼過頭、值得留意的進場訊號。
```

**如果想聚焦頂部訊號：**
```
請特別分析目前是否有過熱或泡沫化的警示訊號。
```
