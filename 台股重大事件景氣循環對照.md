# 台股重大事件景氣循環對照

## 定義

將台股歷史上每次重大回檔事件，對應到 izaax 景氣循環框架的**兩層結構**：超級週期（生產力擴張 vs 通膨擴張）× 小景氣循環（衰退→復甦→成長→榮景）。分析事件發生時的位置，有助於判斷回檔性質（是買點還是大空頭起點）。

## 核心機制

### 兩層框架

```
第一層：超級週期（大週期）
├── 生產力擴張週期：約 20 年（最佳資產：股票）
└── 通膨擴張週期：約 7–10 年（最佳資產：黃金/實體資產）

第二層：在每個超級週期內，有多個小景氣循環
└── 衰退 → 復甦 → 成長 → 榮景 → （再次衰退）…循環數次
```

**關鍵規則（izaax）**：

- 景氣循環按序運行，不跳躍、不逆行
- 4個階段依序出現：衰退 → 復甦 → 成長 → 榮景 → 再次衰退
- 回檔發生在「成長期/復甦期」= 外力衝擊 → 通常是買點
- 回檔發生在「榮景末期」= 循環頂部 → 可能是大空頭起點

### 超級週期二｜通膨擴張（2000–2008，約 8 年）

中國工業化+原物料大牛市，黃金/房地產是主角。

內含一個完整小循環：

```
衰退(2000–02) → 復甦(02–03) → 成長(03–06) → 榮景(06–08) → 大頂
```

| 事件                     | 小循環位置                   | 說明                                                    |
| ------------------------ | ---------------------------- | ------------------------------------------------------- |
| 🟡 **2001/09 911恐攻**   | **衰退期中**                 | 2000 泡沫後已在衰退，911 是衰退期加速衝擊，不改方向     |
| 🟡 **2003 SARS**         | **復甦初期**                 | 這輪通膨週期剛啟動復甦，外力短暫打斷，底部快、反彈強    |
| 🔴 **2008 全球金融海嘯** | **整個通膨週期的大榮景頂點** | 信貸/房市槓桿是通膨週期的最後過熱形式；最典型榮景末崩潰 |

```

小循環 B：衰退(15–16初) → 復甦(16) → 成長(17–18初) → 榮景(18末)
小循環 C：衰退(19) → 人工復甦(20) → 成長(20–21) → 榮景(21末–22初) → 衰退(22)
小循環 D：復甦(23) → 成長(23–24) → 榮景(25–26) ← 目前所在
```

## 規律總結

### 位置 × 回檔性質

| 超級週期位置          | 小循環位置      | 回檔性質               | 操作含義                     |
| --------------------- | --------------- | ---------------------- | ---------------------------- |
| 生產力/通膨週期進行中 | **榮景末**      | 小循環頂部，深但可回   | 等底部信號，不要搶反彈       |
| 生產力/通膨週期終點   | **大榮景頂點**  | 超級週期頂部，最深最長 | 需年以上等待，資產配置大調整 |
| 任何超級週期中        | **成長/復甦期** | 外力截斷，基本面完好   | 通常是最佳買點               |
| 任何超級週期中        | **衰退期中**    | 二次衝擊，加速不改方向 | 持守，不加碼                 |

### 最佳買點類型

**成長/復甦期被外力打斷的事件（歷史驗證均為買點）：**

- 2003 SARS（復甦初，-23%，6個月回原位）
- 2011 311大地震（復甦期，-8%，1個月回）
- 2020 COVID閃崩（成長截斷，-30%，5個月回）
- 2024 日圓Carry Trade（成長期，-19%，2個月回）
- 2025 關稅衝擊（榮景期中，-18%，數日急反）

### 超級週期頂點的特徵（最危險）

| 頂點                                    | 跌幅 | 回本所需時間 |
| --------------------------------------- | ---- | ------------ |
| 1990 台股泡沫（生產力週期內的本土頂點） | -80% | ~16 年       |
| 2000 網路泡沫（生產力週期大頂）         | -67% | ~6 年        |
| 2008 金融海嘯（通膨週期大頂）           | -60% | ~7 年        |

> **超級週期頂部後，回本所需時間都以「年」計算，且通常需要等下一個超級週期啟動。**
